Basic Markdown Tutorial

What is markdown?

Markdown is a simplified markup language, you can think of it as a very simple coding language that allows you to tell the computer how you want a text document to be laid out. Other examples of markup languages are HTML and LateX.


Basic markdown

To type text you simply begin typing. Nothing special needs to be done here (atleast for now). Markdown comes in when we need to style the text in some way (e.g. adding links, italics, numbered lists, bold font etc.).


Italics:

If you wanted the text "this should be in italics" to be in italics, you would type it like this

_this should be in italics_

and the output would be

this should be in italics


Bold:

If you wanted the text "this should be in bold" to be in bold, you would type it like this

**this should be in bold**

and the output would be

this should be in bold


Links:

If you wanted the text "this should be a link to google" to be a link, you would type it like this

[this should be a link to google](https://www.google.com/)

and the output would be

this should be a link to google


Combining:

You can combine the codes above. For example the following code

[**very important link**](https://www.google.com/), **_some text in italics and bold_**

will output this

very important link, some text in italics and bold


Markdown lists

Unordered lists:

This is the code to get unordered lists. Note specifically that there is a space after the "-"

- apples
- bananas
- pears

The output would be

  • item 1
  • item 2
  • item 3

Numbered lists:

This is the code to get unordered lists. Note specifically that there is a space after the "1." and "2." etc.

1. apples
2. bananas
3. pears

The output would be

  1. apples
  2. bananas
  3. pears

Nested lists:

Sometimes you will want to have nested lists where one list item is also a list. This can be done with the following code.Note the two spaces before the "* strawberry" and "* tomato" etc.

1. red fruits
  * strawberry
  * tomato
2. orange fruits
  * mandarin
  * oranges

The output would be

  1. red fruits
    • strawberry
    • tomato
  2. orange fruits
    • mandarin
    • oranges

Markdown headings

You can use bold for headings if you want, but using markdown headings you gives more flexibility and control.

The general idea is fairly simple; add hashes (#) before some text to make it a heading. The more hashes you add the smaller the heading will be. Some code examples are given below.

# Title heading
## Subheading
### Sub-subheading
#### Sub-sub-subheading

The output would be

Title heading

Subheading

Sub-subheading

Sub-sub-subheading
